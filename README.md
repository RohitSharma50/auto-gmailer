# Fullstack Auth (MERN + Redux + Tailwind) with n8n Hook

## Structure

```
backend/
frontend/
```

## Run locally

### Backend

```bash
cd backend
npm install
cp .env.example .env
# set MONGODB_URI, JWT_SECRET, CORS_ORIGIN,
npm run dev
```

### Frontend

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```
