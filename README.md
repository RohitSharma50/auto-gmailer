# Fullstack Auth (MERN + Redux + Tailwind) with n8n Hook

## Structure

```
backend/
frontend/
```
---

## To Clone this Repository

You need to write the following commands on the terminal screen(in vscode) so that you can run this project locally.

```bash
  git clone "https://github.com/RohitSharma50/auto-gmailer"
```

Go to the project directory

```bash
    cd auto-gmailer
```

```bash
    cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

---


## Run locally

### Backend

```bash
cd backend
npm install

# set MONGODB_URI, JWT_SECRET, CORS_ORIGIN,
npm run dev
```

### Frontend

```bash
cd frontend
npm install

npm run dev
```
